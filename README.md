# Clinic Management System

## Overview

This project is a web-based clinic management system designed to help patients and medical staff manage healthcare information in a simple and efficient way.

The main goal of the system is to provide patients with easy access to their medical data, including appointments, prescriptions, and medical visit history. It also allows online appointment booking through an integrated calendar system.

## Key Features

- View upcoming and past medical appointments
- Online appointment scheduling through a calendar system
- Access to prescriptions with the ability to view and print them
- View medical visit history and examination results
- Track progress of treatments and medical visits

## Tech Stack

This project is built using a modern web development stack:

- **Next.js** – frontend and backend framework for building full-stack web applications
- **TypeScript** – for static typing and better code quality
- **Prisma** – ORM for database management and communication
- **Supabase** – database and backend infrastructure (PostgreSQL)
- **Auth0** – authentication and user management system

## Purpose

The system is designed to improve communication between patients and clinics, simplify appointment management, and provide secure access to medical records in one place.
